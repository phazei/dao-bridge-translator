SURGICAL MODE (default — use when the translation is mostly good):

- If an issue is that whole paragraphs or sentences were skipped, restore them by translating the corresponding {source_language} source.

- Do NOT add translator's notes, footnotes, or commentary.
- Output ONLY the corrected {target_language} translation — the full text, with the fixes applied, and nothing else.
